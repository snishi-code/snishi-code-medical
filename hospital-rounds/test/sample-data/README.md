# テスト用サンプルデータ

手動テスト用の**架空データ**です（実在の患者ではなく、個人情報を含みません）。

## comprehensive.device-archive.json
網羅的な「端末まるごと」アーカイブ。2ユーザー・3病棟で、以下を一通り含みます：
- 全ステータス（白／黄／緑／灰／青）
- タグ付き・部屋番号・SOAP・プロブレムリスト・共有欄
- **移動済（転棟）マーカー**の患者
- **外部受信（external）**の患者（再配布制限の確認用）

### 読み込み方
アプリ → 設定（歯車）→ 下にスクロール → **「端末まるごと」→「端末まるごと取込」** → このファイルを選択。
既存データは消えず、`テスト太郎`／`テスト花子` が追加されます（同名ユーザーがいれば合流）。

> 自動テスト（`npm test`）でもこのファイルの妥当性を検査しています（`test/check.mjs` の "sample data"）。
> 中身を変えたらテストも合わせて更新してください。
