# ステータス（色）で進捗を管理する・診察を開始する

患者ボタンの色は **ステータス（進捗）** を表します。色で「誰がまだ／誰が済んだ」を一目で管理できます。

---

## ステータスの色と意味

> 🖼 スクショ：色違いの患者ボタンが並んだホーム画面

色だけでなく **形マーク** も付くので、色覚に関わらず判別できます。

| 色 | マーク | 想定する使い方 | 変更方法 |
| --- | :--: | --- | --- |
| 白 | − | 未回診（初期状態） | デフォルト |
| 黄 | ▲ | 保留・要注意 | 患者画面で選択 |
| 緑 | ✓ | 回診済（QR が表示されます） | 患者画面で選択 |
| 灰 | ✕ | 完了・対象外 | 患者画面で選択 |
| 青 | ★ | 追記・新着（取込時に自動で付与） | データ取込・受信時に自動 |

> 白・青以外は使い方を自由に工夫できます。上記は開発者の想定例です。
> 青は、他端末から取り込んだ患者や、追記された患者に自動で付きます（[06](06_チームで共有する.md) / [08](08_データの保存と復元.md)）。

---

## ステータスを変える

> 🖼 スクショ：ステータス選択ポップアップ（色＋マーク＋名前）

1. 患者画面を開く
2. 上部の **ステータスのマーク** をタップ
3. ポップアップから色を選ぶ

緑にすると、その患者の QR コードが自動生成されます（電子カルテへの転記に使えます → [05](05_電子カルテに転記する.md)）。

---

## 「診察開始」で記録をまとめてリセットする

新しい回診（次の日など）を始めるとき、前回の記録を一括でクリアできます。

> 🖼 スクショ：ホーム画面左上の「診察開始」ボタン

ホーム画面の左上にある **「診察開始」** ボタンを押すと、確認のうえ、**全患者の指定した項目** が一括でクリアされます。

- 何をクリアするかは **設定で選べます**（次項）。
- 患者名・部屋番号・タグなど「残したいもの」はクリア対象から外しておけます。

> 「診察開始」は元に戻せません。不安な場合は先にデータを保存してください（[08](08_データの保存と復元.md)）。

### クリアする項目を設定する

> 🖼 スクショ：設定画面「診察開始でクリアする項目」

設定画面の **「診察開始でクリアする項目」** で、チェックを入れた項目だけがクリアされます。初期設定は次のとおりです。

| 項目 | 初期設定 |
| --- | --- |
| メモ | クリアしない |
| S | クリアする |
| O（バイタル含む） | クリアする |
| A | クリアしない |
| P | クリアしない |
| 共有 | クリアしない |
| ステータス：黄 | クリアする（白に戻す） |
| ステータス：緑 | クリアする（白に戻す） |
| ステータス：灰 | クリアする（白に戻す） |
| ステータス：青 | クリアしない |

> 例：「毎朝、前日の所見とチェック済みステータスは消したいが、プロブレムを書いたメモと申し送り（共有）は残したい」→ S・O・黄・緑・灰のみクリア対象にする（初期設定がこの考え方です）。

---

## すべてのデータを消したいとき

患者・設定を含めて **完全に消去** したい場合は、設定画面のいちばん下にある **「全データを消去する」**（赤いボタン）を使います。元に戻せないため、先にバックアップを取ってください（[08](08_データの保存と復元.md) / [10](10_カスタマイズ（設定）.md)）。
