# カルテ（SOAP）を書く

ホームで患者ボタンをタップすると **患者画面** が開きます。ここで名前・プロブレムリスト・SOAP（S/O/A/P）・共有欄を入力します。

> 🖼 スクショ：患者画面の全体

---

## 患者画面の上部

> 🖼 スクショ：患者画面ヘッダー（患者名ボタン／ステータス／タグ／セット切替）

| 要素              | 役割                                    |
| --------------- | ------------------------------------- |
| 患者名             | タップで名前・部屋番号を編集（[02](02_患者リストを作る.md)）  |
| ステータスのマーク       | タップで色を選択（[04](04_ステータスと診察開始.md)）      |
| タグアイコン          | この患者にタグを付ける（[10](10_タグで分類・絞り込む.md)）   |
| セット切替（`セット名 ▾`） | 表示するフォーマットの「セット」を切り替え（後述）             |
| ？（右上）           | この画面の説明を開く                            |
| QR（右上）          | 入力内容を QR コード化（[05](05_電子カルテに転記する.md)） |

---

## プロブレムリストを書く

> 🖼 スクショ：プロブレムリスト欄（ノートアイコンのカード）

いちばん上の **ノートアイコンのカード** が「プロブレムリスト」欄です。患者を一言で表す **問題リスト・患者サマリ** を書く欄として使います。

- 病名・主病態・申し送りの要点など、**日をまたいで持ち越したい** 情報を入れておきます（「診察開始」では消えません）。
- QR コードを生成したときは、このプロブレムリストが先頭に表示されます（カルテ転記の冒頭）。
- ここに書いた内容は **プロブレムリスト画面**（[06](06_メモ・共有をまとめて見る.md)）にも一覧表示され、名簿QR・プロブレムリストQRで他端末へ渡せます（[07](07_端末間で共有する.md)）。

## S / O / A / P を書く

S・O・A・P それぞれにカードがあり、自由テキストを入力できます。さらに、各欄には **フォーマット** を使った入力補助が付けられます。

### O 欄：バイタルサイン（フォーマット「バイタル」）

> 🖼 スクショ：O 欄に展開されたバイタル入力（BP / P / SpO2 / RR / T）

初期設定では O 欄の上に、バイタル入力欄が **展開表示** されています。

| 項目 | 単位 |
| --- | --- |
| BP | mmHg（収縮期 / 拡張期） |
| P | bpm |
| SpO2 | %（メモ欄あり：例「RA」「カニューレ2L」） |
| RR | （なし） |
| T | ℃ |

数値を入れると、O の出力に `BP 120/80 mmHg, P 70 bpm …` のようにまとめて反映されます。
（バイタル項目の名前や単位は [11_カスタマイズ（設定）](11_カスタマイズ（設定）.md) で変更できます。）

### O 欄：身体所見（フォーマット「身体所見」）

> 🖼 スクショ：身体所見の入力（各項目の「正常」ボタン＋テキスト）

身体所見は **フォーマットのチップ** から開いて入力します。初期設定では次の6項目です。

| 項目 | 「正常」を選んだときの出力 |
| --- | --- |
| General | 良好 |
| 肺音 | 明らかなラ音なし |
| 腸音 | 正常 |
| 腹部 | 平坦軟、圧痛なし |
| 食事 | 摂取良好 |
| 排泄 | 尿・便ともに特記なし |

- 「正常」ボタンを選ぶと定型文（正常文）が入ります。
- 異常があるときは、テキスト欄に所見を入力します。
- 確認していない項目は、何も選ばず空欄のままで構いません。

### S / A / P 欄

S・A・P も O 欄と同じく **展開カード** で入力します。初期設定では「自覚症状（S）」「評価（A）」「方針（P）」の各カードがあり、それぞれ **「正常」ボタン** でよく使う定型文がワンタップで入ります（例：S「特に新しい訴えなし」、A「全身状態は安定」、P「現治療を継続」）。別の内容にしたいときは値をタップして手入力します。

**タップ・入力していない欄は、QR（電子カルテへの転記）には出力されません**（空欄を自動で補う動作はありません）。書かない欄はそのままで構いません。

## フォーマットとセットのしくみ（重要）

このアプリの入力補助は **フォーマット** と **セット** という2段階でできています。

- **フォーマット**：S/O/A/P のどこかに入る「入力テンプレート」。バイタルや身体所見のように、項目（ラベル・単位・正常文）をまとめたもの。
- **セット**：複数のフォーマットを束ねた「組み合わせ」。患者画面右上の `セット名 ▾` から切り替えられます。

各フォーマットは、セットの中で次の2つの出方を選べます。

| 出方 | 説明 |
| --- | --- |
| 展開 | 本文の上に入力カードを常時表示（例：初期設定のバイタル）。ワンタップ入力面 |
| クイック | チップで表示し、タップすると入力モーダルが開く |

> 例えば「発熱対応」「術後」など、状況別のセットを作っておけば、患者ごとに `セット ▾` で切り替えて使えます。
> フォーマット・セットの **作り方** は [11_カスタマイズ（設定）](11_カスタマイズ（設定）.md) を参照してください。

## 共有欄を書く

> 🖼 スクショ：共有欄（人アイコンのカード）

いちばん下の **人アイコンのカード** が「共有」欄です。他の医師・看護師など、別の人へ引き継ぐ情報を書きます。

- 鎮痛薬の処方希望など、これまで口頭やメモ書きで渡していた申し送りを想定しています。
- ここに書いた内容は **共有画面**（[06](06_メモ・共有をまとめて見る.md)）に一覧表示され、QR で他端末へ渡せます。

---

## 入力はいつ保存される？

入力内容は自動で端末内に保存されます。**ただし端末内の保存です。** 端末の初期化やブラウザのデータ消去に備えて、定期的に JSON ファイルとしてバックアップしてください（[09](09_データの保存と復元.md)）。
