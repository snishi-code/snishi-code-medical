# 印刷用ポスター（原案）

> **これは説明書本体とは別の成果物です。**
> 用途：病院が **A4 で印刷して掲示・配布** する1枚もの。スマホ向けの説明書（レスポンシブ HTML）とは目的が違うので、**別レイアウト**で作ります。
> 仕上げ：HTML 化のときに **A4・2列・画像メインの印刷用レイアウト**（印刷 CSS）で組み、「PDF に保存」で出力できるようにします。画面用の説明書と表示が違っても問題ありません（意図的に別物）。
>
> 下書きとして、載せるコピー（文章）とスクショ位置をまとめます。本人が文言調整＋スクショ差し替え後、HTML 化します。

## ヘッダー

- タイトル：**回診アプリ**
- キャッチ：**紙の名簿・メモ帳を、スマホでスマートに。**

## 左列

### インストール（およそ30秒・オフラインで動きます）

1. 下の QR か URL を開く
2. ブラウザのメニューから **「ホーム画面に追加」**（iPhone/iPad は共有ボタンから）
3. 追加されたアイコンから起動。以降はネット不要で使えます

### ⚠️ やってはいけない3つ

- ✕ **個人を特定できる情報を入れない**（フルネーム / 患者 ID / 生年月日 / 住所 / 連絡先 / 画像）
- ✕ **正式な医療記録の代わりにしない**（正式な記録は電子カルテへ）
- ✕ **バックアップを忘れない**（端末初期化等でデータは消えます。設定から JSON 保存）

## 右列

### 3ステップで使う

1. **並べる**：ホームで患者ボタンを長押し → 追加。タップして名前・部屋番号を入力
2. **書く**：患者画面で SOAP・メモを記録（O 欄はバイタル・身体所見の入力補助あり）
3. **送る**：ステータスを緑にすると、電子カルテ転記用の QR が出ます

### 便利ポイント

- 取った所見を **QR で電子カルテに転記**（手打ち不要）
- **完全オフライン／外部送信なし**。データは端末内だけ
- **色で回診の進捗管理**（白＝未／緑＝済 など）
