# 患者リストを作る・並べる・名前を変える

回診の最初のステップは、担当患者をホーム画面に並べることです。

> 🖼 スクショ：ホーム画面（患者がグリッド状に並んだ状態）

ホーム画面では、登録された患者がマス目（グリッド）状に並び、それぞれの **色（ステータス）** で進捗が一目で分かります。色の意味は [04_ステータスと診察開始](04_ステータスと診察開始.md) を参照してください。

> **病棟（ワークスペース）について**：患者リストは「病棟」ごとに独立して管理できます。複数の病棟を切り替えながら使えます。詳しくは [08_病棟と患者の移動](08_病棟と患者の移動.md)。最初は1つの病棟だけで使い始めて問題ありません。

---

## 患者を追加する／削除する（長押しメニュー）

> 🖼 スクショ：患者ボタンを長押しして出る操作メニュー（+1 / −1 / →1 / +5 / −5 / →5）

患者ボタンを **長押しして指を放す** と、操作メニューが開きます。

| ボタン | 操作 |
| --- | --- |
| ＋1 | 選んだ患者の下に1人追加 |
| ＋5 | 選んだ患者の下に5人追加 |
| −1 | 選んだ患者を1人削除 |
| −5 | 選んだ患者から下に5人削除 |
| →1 | 選んだ患者を別の病棟へ移動（[08](08_病棟と患者の移動.md)） |
| →5 | 選んだ患者から5人を別の病棟へ移動（[08](08_病棟と患者の移動.md)） |

- 入力済みデータのある患者を削除しようとすると確認が出ます。
- 新規インストール直後はあらかじめ空の患者枠が用意されています（初期値50枠）。

## 患者名を変える

> 🖼 スクショ：患者画面の上部（患者名ボタン → タップで編集モード）

患者名は **患者画面（詳細画面）** で変更します。

1. ホームで対象の患者ボタンをタップして患者画面を開く
2. 上部の **患者名（色つきのボタン）をタップ** すると編集モードになり、名前の入力欄が出ます
3. 入力して欄の外をタップすると確定します

- 名前が空欄の患者は、ホームのボタンに **番号** が表示されます。
- 名前を入れると、ホームや各一覧のボタンにその名前が表示されます。

> 患者数が多いときは「201 田中」のようにメモページの編集ボタンをクリックするとまとめて編集できます。

## 部屋番号を入力する

部屋番号は、患者ボタンの表示や並び替えに使えます。

> 🖼 スクショ：患者画面の編集モード（患者名の左に部屋番号の入力欄）

### 入力方法

患者画面で患者名をタップして編集モードにすると、**患者名の左に部屋番号の入力欄** が出ます。

- 数字のみ入力できます（英字・記号は自動で除去）
- 最大6桁、空欄も可

メモ画面・共有画面でも、編集モード（鉛筆アイコン）にすると各行に部屋番号欄が出ます。

---

## 患者の並び順

> 🖼 スクショ：部屋番号順に自動で並んだホーム画面

患者は **部屋番号の昇順で自動的に並びます**（各画面を開いたときに整列）。

- 部屋番号がある患者を数値の小さい順に並べる
- 部屋番号が空欄の患者は末尾にまとめられる
- 別の病棟へ移動済みの患者（「(移)」付き）は常に最後尾

> 並び順は部屋番号で自動的に決まります。任意の順番にしたい場合は、部屋番号を使って調整してください。
