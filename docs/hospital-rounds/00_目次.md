## 画面（ビュー）の一覧

ヘッダーのボタンで切り替えます。各画面の右上に共通で **「？」（説明）** と **QR** があり、「？」でこの説明書の該当ページが開きます。

| 画面 | 開き方 | 役割 |
| --- | --- | --- |
| ホーム | ヘッダー左の家アイコン | 患者が一覧（グリッド）で並ぶ起点 |
| 患者画面（詳細） | ホームで患者をタップ | SOAP・プロブレムリスト・共有を入力 |
| プロブレムリスト | ヘッダーのノートアイコン | 全患者のプロブレムリストを一覧・一括編集 |
| 共有 | ヘッダーの人アイコン | 全患者の「共有」欄を一覧・QR で受け渡し |
| 設定 | ヘッダーの歯車アイコン | フォーマット・タグ・病棟管理など |

> 🖼 スクショ：ヘッダー全体（家／ユーザー名▾／病棟名▾ … ノート・人・歯車）

> ヘッダー左の **タイトル枠はユーザー名** になり、タップすると **ユーザー切替**（[08](08_病棟と患者の移動.md)）が開きます。ホームへ戻るのは **家アイコン** です。
