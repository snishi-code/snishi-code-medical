# ホーム

## 概要

![[02_ホーム-1779021052489.webp|300x238]]

起動すると最初に表示される画面です。登録された患者がマス目（グリッド）状に並び、それぞれの色でステータスを一目で確認できます。

---



---

## ステータス

白・青以外は自由に使い方を工夫できます。以下は開発者が想定している一例です。

| 色   | 使用例 | 変更方法         |
| --- | --- | ------------ |
| 白   | 未回診 | デフォルト        |
| 黄   | 保留  | 患者画面から       |
| 緑   | 回診済 | 患者画面から       |
| 灰   | 完了  | 患者画面から       |
| 青   | 追記  | 自動で付く場合もあります |

---

## ボタン説明
左から以下のボタンが並んでいます。

| ボタン  | 機能                                                                 |
| ---- | ------------------------------------------------------------------ |
|      |                                                                    |
| 診察開始 | 患者の一部データを一括で削除。どのデータを削除するかは設定から指定できます。各患者画面からその患者のデータだけを削除することも可能。 |
|      |                                                                    |
|      |                                                                    |

---


タップすると該当患者の詳細入力画面に移動します。

---

## 長押し（患者の追加・削除）

![[02_ホーム-1779021411548.webp|300x238]]

患者ボタンを長押しして放すと操作メニューが表示されます。

| 操作       | 内容                     |
| -------- | ---------------------- |
| 1人追加（下へ） | 選択した患者の下に新しい患者ボタンを1つ追加 |
| 5人追加（下へ） | 選択した患者の下に新しい患者ボタンを5つ追加 |
| 削除       | 選択した患者を削除（確認あり）        |

---
