# データを保存（バックアップ）する・復元する

入力データは端末内にのみ保存されます。**端末の初期化・ブラウザのデータ消去・アプリ更新でデータが失われることがある** ため、JSON ファイルとして定期的に保存（バックアップ）してください。

> 🖼 スクショ：設定画面「病棟」セクションの「JSON 取り込み／JSON 書き出し」ボタン

保存・取込は **設定画面の「病棟」セクション** から行います（ヘッダーの歯車アイコン → 下にスクロール）。

---

## 保存（バックアップ）する

**「JSON 書き出し」** をタップすると、現在のデータが JSON ファイルとしてダウンロードされます。

- ファイル名はタイトル・日付・時刻を含む形式です（例：`回診_2026_0602_1430.json`）。
- **回診終了後や、オンラインにする前に保存することを強く推奨します。**
- Android では自動でダウンロードフォルダに保存されます。iPad / iPhone では「"ファイル"に保存」を選んで保存してください。

> 🖼 スクショ：iPad の「"ファイル"に保存」ダイアログ

---

## 取り込む（復元する）

**「JSON 取り込み」** をタップしてファイルを選びます。取り込むファイルの種類によって動きが変わります。

### バックアップ（複数病棟をまとめたファイル）の場合

> 🖼 スクショ：バックアップ取込の確認ダイアログ

「○個の病棟と設定を取り込みます。既存の病棟は消えず、取り込み分が追加されます」と確認が出ます。**既存の病棟は消えず、取り込んだぶんが追加** されます。

### 1つの病棟ぶんのファイルの場合

> 🖼 スクショ：取込の選択（設定も取り込む／患者のみ取込）

現在の病棟に取り込みます。次のどちらかを選べます。

| 選択 | 動き |
| --- | --- |
| 設定も取り込む | 患者を末尾に追加し、フォーマット・タグなどの設定も取り込む |
| 患者のみ取込 | 患者だけを末尾に追加（設定は変更しない） |

- 取り込まれた患者は末尾に追加され、ステータスは自動で **青（★）** になります。
- 取り込みで既存の患者・設定が消えることはありません（追加されます）。

> **元に戻したいとき** に備えて、取り込みの前にも現状を「JSON 書き出し」で保存しておくと安心です。

---

## 全部消したいとき

設定画面のいちばん下の **「全データを消去する」**（赤ボタン）で、すべての病棟・設定を完全に削除できます。元に戻せないため、必ず先にバックアップしてください。
