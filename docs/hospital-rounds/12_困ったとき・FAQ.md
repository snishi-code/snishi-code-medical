# 困ったとき・よくある質問（FAQ）

## いちばん大事な注意

- **個人が特定できる情報は入力しない**：フルネーム / 患者 ID / 生年月日 / 住所 / 連絡先 / 画像（[01](01_クイックスタート.md)）。
- **本アプリは正式な医療記録ではありません**。正式な記録は電子カルテに残してください。
- **データは端末内のみ**に保存されます。外部送信はありません。そのぶん **バックアップは自己責任** です（[09](09_データの保存と復元.md)）。
- 医療機関での導入は、施設の承認を得てから行ってください。

---

## データを失わないために

- **こまめに JSON 保存**（[09](09_データの保存と復元.md)）。Word の上書き保存と同じ感覚で。
- **オンラインにする前に保存**：PWA はオンライン時に更新されることがあり、更新時にデータ破損のリスクがあります（仕組みは [13](13_システム担当者向け.md)）。
- **「診察開始」「全データを消去」の前に保存**：いずれも元に戻せません（[04](04_ステータスと診察開始.md)）。

## よくある質問

**Q. 操作したのに画面の表示が変わりません。**
A. 別の画面に移動して戻ると反映されることがあります。多くの操作は自動で反映されますが、うまくいかないときは画面を切り替えてみてください。

**Q. 患者を好きな順番に並べたいです。**
A. 並び順は **部屋番号の昇順で自動** に決まります。任意の順にしたい場合は部屋番号で調整してください（[02](02_患者リストを作る.md)）。

**Q. 患者ボタンが青（★）になりました。**
A. 他端末から取り込んだ・受信した・移動した患者に自動で付く色です（[04](04_ステータスと診察開始.md) / [07](07_端末間で共有する.md)）。手動で別の色に変えられます。

**Q. QR が複数枚出ます。**
A. データ量が多いと自動で分割されます。左右の矢印で切り替えて、すべて読み取ってください（[05](05_電子カルテに転記する.md) / [07](07_端末間で共有する.md)）。

**Q. 他端末の QR が読み取れません。**
A. QR は暗号化されており、原則 **同じバージョンのアプリどうし** で読み取れます。送受信で版を揃えてください。

**Q. S / A / P によく書く定型文を自動で入れたいです。**
A. 「規定文」付きのフォーマットを作り、デフォルトのセットに入れてください（[11](11_カスタマイズ（設定）.md)）。

**Q. バイタルの項目名や単位を変えたいです。**
A. 設定でフォーマット「バイタル」を編集すれば変更できます（[11](11_カスタマイズ（設定）.md)）。

**Q. 以前のバージョンに戻したいです。**
A. [GitHub](https://github.com/snishi-code/snishi-code.com) から任意のバージョンに戻せます。システム担当者に確認してください。

## 用語まとめ

| 用語           | 意味                                                               |
| ------------ | ---------------------------------------------------------------- |
| 病棟           | 独立した患者リストの単位。切替・新規作成・移動ができる（[08](08_病棟と患者の移動.md)）                |
| フォーマット       | S/O/A/P に入る入力テンプレート（[03](03_カルテを書く.md) / [11](11_カスタマイズ（設定）.md)） |
| セット          | フォーマットの組み合わせ。患者ごとに切替（[03](03_カルテを書く.md)）                         |
| 規定文          | 欄が空のときに出力へ補う定型文                                                  |
| ステータス        | 患者ボタンの色（進捗）。白／黄／緑／灰／青（[04](04_ステータスと診察開始.md)）                    |
| 診察開始         | 設定した項目を全患者で一括クリアする操作（[04](04_ステータスと診察開始.md)）                     |
| 共有 QR / 受信ボックス | 端末間で情報を受け渡すしくみ（[07](07_端末間で共有する.md)）                             |
