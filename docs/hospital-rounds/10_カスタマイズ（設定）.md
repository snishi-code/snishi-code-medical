# カスタマイズする（設定画面）

ヘッダーの **歯車アイコン** で設定画面を開きます。入力欄やフォーマットを自分の使い方に合わせて調整できます。

> 🖼 スクショ：設定画面の全体

設定画面には上から次のカードが並びます。

| カード | 内容 | 詳細 |
| --- | --- | --- |
| S / O / A / P | 各欄のフォーマット | このページ |
| セット | フォーマットの組み合わせ | このページ |
| 診察開始でクリアする項目 | 一括クリアの対象 | [04](04_ステータスと診察開始.md) |
| タグ | タグの管理 | [09](09_タグで分類・絞り込む.md) |
| タイトル | アプリ名（端末ごと） | このページ |
| 病棟 | 病棟の名前変更・削除・JSON 取込/保存 | [07](07_病棟と患者の移動.md) / [08](08_データの保存と復元.md) |
| 全データを消去する | すべて削除 | [08](08_データの保存と復元.md) |

---

## アプリのタイトルを変える

> 🖼 スクショ：設定画面の「タイトル」欄

**「タイトル」** の入力欄で、ヘッダーに表示されるアプリ名（初期値「回診」）を変更できます。担当者名や用途を入れて使い分けられます（この設定は端末ごとに保持されます）。

---

## フォーマットを作る・編集する

フォーマットは、S/O/A/P のいずれかに入る **入力テンプレート** です（バイタル・身体所見など。しくみは [03](03_カルテを書く.md)）。

> 🖼 スクショ：O カードのフォーマット一覧と「＋」

1. S/O/A/P いずれかのカード右上の **「＋」** をタップ（その欄に属するフォーマットを追加）
2. 既存のフォーマットは **鉛筆** で編集、**ゴミ箱** で削除

### フォーマット編集の中身

> 🖼 スクショ：フォーマット編集モーダル（名前・タグ・項目）

| 設定  | 内容                                                               |
| --- | ---------------------------------------------------------------- |
| 名前  | フォーマット名（例：バイタル）。同名は不可                                            |
| タグ  | このフォーマットに紐づけるタグ。例えば、Pで「検査予定」フォーマットを使用すると、自動で「検査予定」タグを付けるなどが可能です。 |
| 項目  | 入力する項目を並べる。「＋ 項目追加」で追加                                           |

各項目には **様式** を選べます。

| 様式      | 用途               | 例             |
| ------- | ---------------- | ------------- |
| 文字（Aa）  | テキスト。「正常文」を設定できる | 身体所見の各項目      |
| 数値（123） | 数値＋単位            | 体温（℃）、SpO2（%） |
| 分数（a/b） | 分数形式＋単位          | 血圧（mmHg）、日付など |

- **正常文**：文字様式で、「正常」ボタンを押したときに入る定型文。
- **単位**：数値・分数様式で、値の後ろに付く単位。

> モーダル左下の **QR アイコン** で、このフォーマットを QR にして他端末へ渡せます（フォーマット QR）。先に保存してから共有してください。

---

## セットを作る（フォーマットの組み合わせ）

セットは、複数のフォーマットを束ねた「組み合わせ」です。患者画面の `セット ▾` で切り替えて使います（[03](03_カルテを書く.md)）。

> 🖼 スクショ：セットカードと「＋」／セット編集モーダル

1. **セット** カード右上の **「＋」** で新規作成、既存は鉛筆で編集
2. 編集モーダルで設定する内容：

| 設定 | 内容 |
| --- | --- |
| 名前 | セット名（例：発熱対応） |
| このセットをデフォルトにする | 患者でセット未選択のとき使われるセット（必ず1つ） |
| 含めるフォーマット | このセットに入れるフォーマットを選ぶ |

各フォーマットは、セットの中で **展開／クイック／規定文** の出方を選べます（意味は [03](03_カルテを書く.md)）。

- **展開**：本文の上に入力欄を常時表示
- **クイック**：チップで表示、タップで入力モーダル
- **規定文**：欄が空のとき、その内容を出力に補う（例：A 欄が空なら「著変なし」を補う、など）

> 「規定文」を使うと、旧版の「S・A・P の空欄時デフォルト文言」と同じことができます。よく書く定型文を規定文付きのフォーマットにして、デフォルトのセットへ入れておきましょう。

---

## 設定を他端末にそろえる（設定 QR）

> 🖼 スクショ：設定画面右上の QR 表示

設定画面右上の **QR アイコン** で、タグ・フォーマット・クリア対象などの設定を QR にできます。別端末でカメラ／貼り付けから読み取ると、その端末の設定を上書きできます（端末固有の設定は維持されます）。
