# クイックスタート

> このページは「最初の1分」のための最小ガイドです。**基本的には触れば使えます。** 各画面の右上の **「？」** を押すと、その画面の詳しい説明が開きます。

## これは何？

回診時の患者情報を、スマホ・タブレットで手軽に記録するアプリです。紙の名簿やメモ帳の置き換えを目的としています。**正式な医療記録は電子カルテに記載してください。**

## ⚠️ 使う前に（3つだけ）

1. **個人が特定できる情報は入力しない** — フルネーム / 患者 ID / 生年月日 / 住所 / 連絡先 / 画像 / 特殊な所見 / 詳細な病歴。部屋番号や一般的な所見など、個人を特定しないものだけにしてください。
2. **これは個人メモ** — 正式な記録ではありません。データは端末内にのみ保存され、外部には送信されません。
3. **こまめにバックアップ** — 端末の初期化等でデータは消えます。設定画面から JSON 保存をしておくと安心です（[08](08_データの保存と復元.md)）。

## 3ステップで使う

> 🖼 スクショ：ホーム → 患者画面 → 緑ステータスで QR、の3枚を並べる

1. **患者を追加** — ホームで患者ボタンを長押し → 追加。タップで開いて名前・部屋番号を入力（[02](02_患者リストを作る.md)）。
2. **カルテを書く** — 患者画面で SOAP・メモを入力。O 欄にはバイタル・身体所見の入力補助があります（[03](03_カルテを書く.md)）。
3. **色で管理する** — 回診済みは緑など、色で進捗を管理。（[04](04_ステータスと診察開始.md) / [05](05_電子カルテに転記する.md)）。

翌日など、記録をまとめて消して回診をやり直すには、ホーム左上の **「診察開始」** を使います（[04](04_ステータスと診察開始.md)）。
